# FOREST AREA VISUALIZE

Developed with React Three.js p5.js

## 開発環境
- node v16.13.1
- yarn
- React
- Three.js
- p5.js

## インストール
```
$ yarn install
```

## 開発手順
```
$ yarn start
```

## ビルド
```
$ yarn build
```

## フォルダ構成
- client/public
  - html,画像データ等
- clirnt/src
  - /canvas - Three.js & p5.js クラスファイル
  - /scenes - React html
